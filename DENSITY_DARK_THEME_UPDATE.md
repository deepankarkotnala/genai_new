# Desktop density and dark theme update

Updated `genai-portal/assets/styles.css` to:

- loosen the hero heading tracking so “Learn GenAI engineering…” no longer looks cramped;
- reduce desktop-only typography, navigation chrome, hero padding, card padding, gaps and radii;
- widen the reading area slightly so content wraps less and more is visible per screen;
- preserve the existing mobile sizing and touch layout;
- refresh dark mode with a deeper navy base, layered glass surfaces, cool violet/blue highlights, subtler borders and clearer text contrast.

The dark treatment takes general inspiration from the referenced portfolio’s compact, layered-glass design language without copying its layout or source styles.
