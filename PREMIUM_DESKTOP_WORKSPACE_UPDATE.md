# Premium desktop workspace update

## Changes

- Reduced the desktop command bar to a 42 px glass strip with smaller controls.
- Reduced SVG icon stroke weight and icon dimensions for a cleaner professional appearance.
- Narrowed the desktop sidebar and table-of-contents rail.
- Reduced sidebar typography, padding, row height, search height, and brand size for denser navigation.
- Expanded the primary content canvas so hero panels, cards, diagrams, and learning content use more desktop width.
- Refined borders, shadows, radii, and glass gradients across desktop surfaces.
- Kept mobile drawer sizing and touch-target behavior unchanged.
