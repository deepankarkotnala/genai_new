# Reader controls fix

The Reader menu now changes lesson typography, readable line length and contrast in a visible, persistent way.

- Text size applies to prose, headings, cards, tables, code, quizzes and demos.
- Line width changes the actual article measure on screens with enough room.
- High contrast strengthens text, borders and glass surfaces in light and dark themes.
- Controls expose pressed state and announce changes to assistive technology.
- Settings remain stored locally between pages.
